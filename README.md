# Simulating-the-Universe-using-Diffusion-Models

Dataset:
CAMELS Multifield Dataset (~15,000 samples)

Models:
- CNN for cosmological parameter regression
- Diffusion models (DDPM/DDIM) for hydrodynamical map generation

Results:
- Ωm prediction error: ~4%
- σ8 prediction error: ~2%

The full methodology and results are documented in the report.
